Web Accessibility Simulation.

Allows developers to simulate the perspective of a user with:
- Colour-blindness.
- Loss of motor functions (Keyboard navigation only).
- Partial blindness.

Before running this file, please add an "id = body" to your body element. (e.g. <body id="body">)

To run the buildscript.js file on your HTML then please write this line of code:
- <script type="text/javascript" charset="utf-8" src="buildscript.js"></script>
